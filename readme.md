# Typescript

A comprehensive collection of TypeScript code examples covering fundamental to advanced concepts. This repository serves as a practical reference and learning resource for TypeScript.

## Contents

This repository covers the following TypeScript concepts:

- **Basic Types** - String, Number, Boolean, Array, Tuple, etc.
- **Type Annotations** - Explicit type declarations
- **Type Inference** - Automatic type detection
- **Interfaces** - Object shape definitions
- **Enums** - Named constants
- **Generics** - Reusable type-safe components
- **Classes** - Object-oriented programming
- **Access Modifiers** - public, private, protected
- **Abstract Classes** - Base class definitions
- **Functions** - Type-safe function declarations
- **Union & Intersection Types** - Flexible type combinations
- **Type Guards** - Runtime type checking

## Purpose

This repository contains hands-on code examples for each TypeScript concept and best practices.

## Getting Started
```bash
# Clone the repository
git clone https://github.com/yourusername/Typescript.git

# Navigate to the project
cd Typescript

# Compile TypeScript files
tsc filename.ts

# Run compiled JavaScript
node filename.js
```

## Usage

Browse through the files to find examples of specific TypeScript concepts. Each file contains working examples with explanatory comments.

To compile and run any example:
```bash
tsc example.ts && node example.js
```

## Prerequisites

- Node.js
- TypeScript (`npm install -g typescript`)
